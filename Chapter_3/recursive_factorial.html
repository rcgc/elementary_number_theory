Definition

Is the product of the first n positive integers, that is, n = n × n−1 × n−2 × ... ×3×2×1

Pseudocode

// Will return the facotrial of a given number

PROCEDURE FACTORIAL( n ):
IF( n ≤ 1 ):
RETURN 1
ELSE:
RETURN n * FACTORIAL( n − 1 )
